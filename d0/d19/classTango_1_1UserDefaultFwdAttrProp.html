Detailed Description

User class to set forwarded attribute default properties.

This class is used to set forwarded attribute default properties. Three levels of attributes properties setting are implemented within Tango. The highest property setting level is the database. Then the user default (set using this UserDefaultFwdAttrProp class) and finally a Tango library default value
